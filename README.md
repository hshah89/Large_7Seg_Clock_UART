## Commands

To set the time
```
time,year,month,day,hour,minute,second
```

To set the time color
```
color_time,red,green,blue

red,green,blue values = 0 to 255
```


To set the temp color
```
color_temp,red,green,blue

red,green,blue values = 0 to 255
```

Save the colors to the EEPROM
```
save,
```
